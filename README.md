# Betting App

A mobile-oriented betting interface with an interactive confidence level slider.

## Features

- **Adjustable Bet Amount**: Edit bet amount directly
- **Interactive Confidence Slider**: iOS-style slider with visual feedback
- **Dynamic Calculations**: Real-time "To Win" amount based on confidence level
- **Market Confidence Indicator**: Shows current market confidence with visual dashed line
- **Responsive Design**: Optimized for mobile devices

## Formula

To Win = Bet Amount รท Confidence Level

## Tech Stack

- Pure HTML/CSS/JavaScript
- Express.js (for serving)
- No build process required

## Local Development

```bash
# Install dependencies
npm install

# Start server
npm start

# Open browser to http://localhost:3000
```
